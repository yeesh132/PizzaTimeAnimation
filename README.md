# Pizza Time Animation

## Project Overview
This project is a cinematic 3D animation created using the Unity Engine, showcasing a commercial for a failing pizza place. The animation has multiple settings across its five scenes, featuring cinematic camera work, lighting effects, smoke particles, sound effects, and character reactions.

## Software Used
- Unity 2022.3.62f3 LTS
- Cinemachine
- Unity Timeline

## Video Link
https://www.youtube.com/watch?v=-BwBzlrxKTQ

## Features
- Cinematic camera transitions
- Garage door opening sequence
- Vehicle reveal animation
- Character reaction animations
- Dynamic lighting
- Smoke particle effects
- Background music and sound effects
